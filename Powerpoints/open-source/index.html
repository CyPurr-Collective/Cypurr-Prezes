What is Open Source?

History of Open Source

  • Started in 1979 with TeX
  • Continued in 1983 by Richard Stallman with GNU
  • ~1994 Netscape Navigator is King
  • 1995 Microsoft bundles Internet Explorer with Windows 95
  • Instead of folding, Netscape releases its code for free, becomes Mozilla foundation

Why Open Source?

Design Principles

  • Users are co-developers
  • Early and frequent releases
  • Multiple versions
  • Modularity (code pieces can be reused)
  • Dynamic Decision Making
    • Average OSS project has 2-5 developers
  • Can focus on 'best' code rather than 'cheapest' solution

Open Source Makes the Web Go

 Almost every website uses an open source software called Linux

  

Protocols that make the internet work (like http and html) are also open source

 

Knowing how other parts of the internet work allows you to extend that functionality
